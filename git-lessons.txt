Створення репозиторію:
Знаходячись у папці вашого проекту виконайте наступну команду:

git init 

вона створить у вашій поточній папці приховану папку .git в якій будуть зберігатись всі данні про ваш проект які гіт запамятає
Щоб вивести собі довідку про поточний стан репозиторію ви можете ввести команду 

git status
